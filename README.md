# สานศิลป์ไทย (Sann-Silp-Thai)

[English Summary Below]

เว็บไซต์สานศิลป์ไทยแห่งนี้ ถูกพัฒนาขึ้นเพื่อใช้สำหรับเผยแพร่และนำเสนอ 🇹🇭 **Soft Power ของไทยในมิติของ "งานจักสาน"** ซึ่งเป็นภูมิปัญญาและศิลปหัตถกรรมที่สะท้อนถึงประวัติศาสตร์ วิถีชีวิต และความใกล้ชิดกับธรรมชาติของคนไทย โครงการนี้ได้รับการออกแบบในรูปแบบมัลติเพจ (Multi-page Architecture) เพื่อบรรจุเนื้อหาและข้อมูลเชิงลึกทางวิชาการได้อย่างสมบูรณ์

## จุดเด่นของโปรเจกต์ (Key Features)

*   **Pure Vanilla Stack:** พัฒนาขึ้นโดยใช้เพียง HTML, CSS, และ JavaScript 100% **โดยไม่มีการพึ่งพา Framework หรือ Library ใดๆ สำเร็จรูป** (เช่น Bootstrap, Tailwind, jQuery) เพื่อรีดประสิทธิภาพและความคล่องตัวขั้นสุด
*   **Custom Preloader:** ระบบโหลดหน้าเว็บที่ถูกเขียนขึ้นเองผ่าน SVG เป็นรูป **ปลาตะเพียนสาน** ซึ่งจะทำงานทุกครั้งที่มีการเปลี่ยนหน้าเว็บ
*   **Grid System แบบกำหนดเอง (Custom Grid):** วางโครงสร้าง Layout และ Component ต่างๆ ขึ้นมาโดยอาศัย CSS Flexbox และ Grid
*   **ความสมบูรณ์ของเนื้อหาเชิง Academic:** รวบรวมข้อมูลตั้งแต่กำเนิด, การใช้วัตถุดิบทั้ง 4 ภาค (ไผ่รวก ไผ่สีสุก ย่านลิเภา กระจูด), ไปจนถึงเจาะลึกภาชนะชนิดต่างๆ (ก่องข้าว งอบ กระติบ กระบุง) แบบครบถ้วน ไม่ตกหล่นแม้แต่บรรทัดเดียว
*   **Typography:** ใช้ฟอนต์ **IBM Plex Sans Thai Looped** เป็นหลักเพื่อความอ่านง่าย สวยงาม และเข้ากับเอกลักษณ์งานออกแบบอย่างไทย

## โครงสร้างและการนำทาง (Site Architecture)

เว็บไซต์ถูกแบ่งออกเป็น 5 หน้าหลัก ดังนี้:

1.  `index.html` (หน้าแรก): หน้า Landing Page นำเสนอภาพรวมของโปรเจกต์ นำเข้าสู่แง่มุมความสวยงามของจักสานไทย
2.  `history.html` (ภูมิปัญญาจากเส้นใย): ประวัติความเป็นมาตั้งแต่สมัยก่อนประวัติศาสตร์บ้านเชียง ประเภทของมีด เครื่องมือจักสาน และลายสานประเภทต่างๆ
3.  `regions.html` (อัตลักษณ์แห่งภูมิภาค): เจาะลึกความแต่งต่างในรายละเอียดของงานจักสานประจำภาค ทั้งภาคเหนือ กลาง อีสาน และใต้
4.  `soft-power.html` (เส้นทางสู่ Soft Power): การประยุกต์งานหัตถกรรมให้กลายเป็นแฟชั่นระดับโลก และมูลค่าเชิงเศรษฐกิจ
5.  `save-craft.html` (สานต่อวิถีไทย ให้ยั่งยืน): แหล่งสืบค้นข้อมูลต่อยอด เช่น สารานุกรมไทยสำหรับเยาวชนฯ, SACIT รวมถึงส่วนแนะนำผู้จัดทำ (About Us)

## วิธีการรันโปรเจกต์ (How to Run)

เนื่องจากโปรเจกต์นี้เขียนด้วย Vanilla HTML ล้วน จึงไม่จำเป็นต้องติดตั้ง Dependencies ใดๆ
1. ดาวน์โหลดหรือโคลน (Clone) Repository นี้
2. ดับเบิลคลิกเปิดไฟล์ `index.html` ด้วยเว็บเบราว์เซอร์ใดก็ได้ (Google Chrome, Safari, Firefox, Edge) ทันที

### English Summary

**Sann-Silp-Thai** is a multi-page web platform dedicated to promoting Thailand's Soft Power through the lens of traditional woven handicrafts. Developed entirely using **pure HTML, CSS, and JavaScript (Zero Frameworks)**, the website features custom CSS grid implementations, a dynamically built SVG "Pla Tapian" (woven fish) preloader, and meticulously detailed academic-level content on Thai craftsmanship. The project contains comprehensive documents detailing regional stylistic differences, material foraging (bamboo, rattan, ferns), and evolutionary history, bringing traditional Southeast Asian craft documentation to the modern web.
